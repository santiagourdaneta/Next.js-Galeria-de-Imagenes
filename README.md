El proyecto está diseñado para demostrar habilidades en el desarrollo de aplicaciones web full-stack, desde la base de datos hasta la interfaz de usuario.

## Características ✨

-   **Galería Interactiva**: Explora todos los dibujos en un diseño de cuadrícula moderno (estilo Masonry).
-   **Sistema de Carga de Imágenes**: Sube tus propias imágenes.
-   **Tecnología Moderna**: Construido con Next.js y un sistema de base de datos local usando SQLite.
-   **Diseño Responsivo**: Se ve bien en computadoras y dispositivos móviles.

## Tecnologías Utilizadas 🚀

-   **Frontend**: [Next.js](https://nextjs.org/) (React)
-   **Backend**: [Next.js API Routes](https://nextjs.org/docs/api-routes/introduction)
-   **Base de Datos**: [SQLite](https://www.sqlite.org/index.html) (a través de `better-sqlite3`)
-   **Estilos**: CSS (con un enfoque en flexbox y grid para el diseño)

## Cómo Empezar 🏁

Sigue estos pasos para ejecutar el proyecto en tu máquina local.

**1. Clona el repositorio:**

```bash
git clone https://github.com/santiagourdaneta/Next.js-Galeria-de-Imagenes/
cd Next.js-Galeria-de-Imagenes

Instala las dependencias:

npm install

Configura la base de datos y los datos de prueba:

# Borra la base de datos si existe y crea una nueva con 250 registros
node seed.js

Inicia el servidor de desarrollo:

npm run dev

Abre http://localhost:3000 en tu navegador para ver la aplicación.
